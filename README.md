# Real Estate Price Predictor API Prep

## Project Overview
This project processes large-scale housing transaction data using Python and Pandas.

## Features
- Chunk-based CSV loading for large datasets
- Exploratory Data Analysis (EDA)
- Outlier detection and removal
- Feature engineering
- Parquet file generation

## Technologies Used
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- PyArrow
- Google Colab

## Engineered Features
1. Age of Home
2. Price Per Square Foot
3. Distance to City Center

## Output
Cleaned dataset saved in `.parquet` format for faster ML processing.
